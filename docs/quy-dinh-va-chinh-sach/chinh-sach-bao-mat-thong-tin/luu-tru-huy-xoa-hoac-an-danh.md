# 6. Lưu trữ, hủy, xóa hoặc ẩn danh

Chúng tôi chỉ lưu giữ Dữ liệu cá nhân của Bạn trong khoảng thời gian cần thiết để hoàn thành mục đích thu thập dữ liệu đó. Sau đó, Chúng tôi tìm cách hủy, xóa vĩnh viễn hoặc ẩn danh dữ liệu đó.
